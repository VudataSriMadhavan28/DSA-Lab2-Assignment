Part B: Doubly Linked List

Displayed list in forward (next) and backward (prev) directions.

Methods Used

Dynamic memory allocation using malloc() and free().
Pointer manipulation for linking nodes.
Traversal techniques for operations.

Time Complexity

Most operations: O(n)

Conclusion These programs help understand linked list concepts, pointer usage, and common DSA operations in an exam-oriented way.
